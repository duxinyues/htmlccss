# 浮动布局

### 原理

在网页布局中浮动就是将一个元素拉到它容器的一侧，让文档能够包裹它。尤其是在图片和文字的排版，图片移动到网页的一侧，并且让文字围绕图片的效果，这时浮动布局是唯一的选择。

```css
main {
            background-color: #fff;
            padding: 0 1.5em;
            border-radius: .5em;
            float: left;
            border: 1px solid red;
        }

        main::after {
            display: block;
            content: "";
            clear: both;
        }

        .content {
            border: 1px solid red;
            width: 500px;

        }

        .clear {
            clear: both;
            border: 1px solid red;
        }
```
```css
    <main>
        <div class="item">1</div>
        <div class="item">2</div>
        <div class="item">3</div>
        <div class="item">4</div>

    </main>

    这种布局常用于将网页内容居中。通过将内容放置到两个嵌套的容器中，然后给内层的容器设置外边距，让它在外层容器中居中（如图4-5所示）。Web开发人员Brad Westfall把这种布局方式叫作双容器模式（double
    container pattern）。

    <div class="clear"></div>
    <div class="content">content </div>
```
